A fast implementation of a two-dimensional ordering transformation for arbitary lenghts (works for all sizes, not just power of 2) This does not perform a complete wavelet transformation. It just does the cascade ordering of the values. The implementation takes advantage of multiple CPU cores.

Parameters:
widthThe width of the transformation
heightThe width of the transformation
